# Usability Test Plan

## Goal

Evaluate the shopping application's usability.

---

## Target Users

Age: 18-40

Shopping Experience:
- New users
- Existing users

---

## Devices

- Mobile
- Desktop

---

## Test Environment

Remote usability testing

---

## Tasks

1. Sign Up
2. Login
3. Search Product
4. View Product
5. Add to Cart
6. Checkout
7. Track Order

---

## Success Criteria

- Complete tasks without help
- Minimal errors
- Completion within expected time
