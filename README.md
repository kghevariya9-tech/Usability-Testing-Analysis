# Usability Testing & Analysis

## Project Overview

This project demonstrates the complete usability testing process for a digital shopping application.

The objective was to identify usability issues, evaluate user experience, and recommend improvements based on user feedback.

---

## Objectives

- Prepare usability test plan
- Create user scenarios
- Conduct usability sessions
- Record user observations
- Analyze usability metrics
- Suggest UI improvements

---

## Tools Used

- Figma
- Google Forms
- Microsoft Excel
- GitHub

---

## Testing Method

- Moderated usability testing
- Think-Aloud Method
- Observation
- User Feedback

---

5 Sample Users

Experience:
- Beginner

- Advanced
